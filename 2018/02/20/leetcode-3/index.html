leetcode 3 Longest Substring Without Repeating Characters

给一串字符串，找出字符串中最长的没有重复的部分，给出长度。

思路：设置一个计数器count和指示器k。将字符串放入map中，边放入，边查找放入的字符在map中是否有相同字符，若没有计数器加1，若有相同的字符串，有两种情况，一是相同字符的位置在指示器之后，则直接计数器加1，二是相同字符位置在指示器之后，则将指示器设置为相同字符的下一位置，同时设加入的字符串是第i位则计数器设置为i-k+1。

刚开始做时，我用一个大数组存放每一个没有重复字符段字符的个数，然后找出数组中的最大值。但是随着字符串变长，算法耗时更多。leetcode报超时错误。代码如下:

之后改进为在计数途中就不断的判断最大值,代码如下：

class Solution {
    public int lengthOfLongestSubstring(String s) {
        int count=0,max=0;
        int k=0;
        Map<Character,Integer> map = new HashMap<Character,Integer>();
        for(int i=0;i<s.length();i++){
            if(!map.containsKey(s.charAt(i))){
                count++;
                max=Math.max(count,max);
            }else{
                if(map.get(s.charAt(i))<k){
                    count++;
                   max=Math.max(count,max);
                }else{
                    k=map.get(s.charAt(i))+1;
                    count=i-k+1;
                    max=Math.max(count,max);
                }
            }
            map.put(s.charAt(i),i);
        }
        return max;
    }
}
